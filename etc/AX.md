# 1. doc tool(with Vector DB)
- 기본적으로 pdf 대상 업로드
- 일반 문서 파일의 경우 폰트 파일이 서버 사이드에 부재하면 문서 데이터가 깨질 수 있음

# 2. DB tool
- DB 메타데이터와 함께 업로드 필요(메타 테이블, 메타 컬럼, 메타 릴레이션쉽 정보)
- 메타 데이터에 따라 능동적으로 Agent가 데이터 간의 상관관계, 맥락 파악하여 쿼리 생성

# 3. 소스 실행 도구
- 소스 코드를 데이터베이스화 하여 사용
- 소스 퀄리티가 떨어지면(너무 길거나 객체화가 잘 되어 있지 않은 경우)에는 당연히 성능이 떨어질 수 있음
